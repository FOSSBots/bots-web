Deploying new code

Pushing to dev

Create a new branch with your changes and then create a PR to push to 'dev' Push the changes to the 'dev' branch Restart the bot

Test your change

Test your change and keep updating the branch for change Push every change from your new branch to 'dev'

Push to the live bot

Push your change to the 'master' branch from the branch created for your change Restart the bot and test nothing else has broken

Update working branches

Update all other braches with the new change Delete the working branch you created for your change Your change is now deployed across the cluster and on all development branches.
